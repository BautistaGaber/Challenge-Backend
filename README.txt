# Challenge-Backend

README 
Proyecto de API REST.
En este proyecto implementé el modelo de Code First para modelar la base de datos. El modelo de dominio esta compuesto por Character, Movie, Gender y una clase de relacion de tablas Many to many llamada CharacterMovie.
Luego en la base de datos cargue 5 valores en la tabla de Genero para utilizarlos dentro del proyecto ya que, para crear una serie y asignarle un genero, este ya debe existir.
También cree la autenticación de usuario, tanto como para registrarse y logearse. y valide que no se pueda registrar con el mismo nombre de Usuario.
